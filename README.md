## 📋 Description

**Niger Laptop** est une plateforme e-commerce complète développée pour la vente de produits informatiques et technologiques au Niger. Le site arbore fièrement les couleurs du drapeau nigérien : **Orange** 🇳🇪, **Blanc** et **Vert**, symbolisant l'engagement de l'entreprise envers le marché local.

## ✨ Fonctionnalités

### 🛍️ Gestion des produits
- Affichage dynamique des produits avec catégories
- Système de badges (PROMO, NOUVEAU)
- Notation par étoiles (avis clients)
- Prix original et prix promo
- Recherche de produits en temps réel

### 🛒 Panier d'achat
- Ajout/retrait de produits
- Modification des quantités
- Calcul automatique du total
- Persistance des données avec localStorage
- Interface coulissante (sidebar)

### 📱 Interface responsive
- Design adaptatif (mobile, tablette, desktop)
- Menu burger pour mobile
- Grille de produits flexible

### 📄 Pages incluses
- **Accueil** : Hero section, catégories, produits vedettes, newsletter
- **Produits** : Catalogue complet avec filtres
- **À propos** : Informations légales et présentation
- **Contact** : Formulaire de contact avec coordonnées

### 🎨 Caractéristiques techniques
- Animations CSS fluides (fadeIn, slideIn, pulse)
- Notifications toast pour les actions utilisateur
- Modal de confirmation de commande
- Barre de recherche intelligente
- Footer complet avec liens et réseaux sociaux
